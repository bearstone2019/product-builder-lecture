
# 청사진: 로또 번호 생성기

## 개요

이 프로젝트는 웹 기반 로또 번호 생성기입니다. 사용자에게 무작위 로또 번호를 생성하는 재미있고 상호작용적인 방법을 제공합니다. 이 애플리케이션은 최신 프레임워크 없는 웹 기술(HTML, CSS, JavaScript)을 사용하여 구축되며, 깔끔하고 시각적으로 매력적이며 반응형 디자인을 특징으로 합니다.

## 디자인 및 기능

### 시각적 디자인 (미학)
- **테마:** 현대적이고 재미있는 느낌.
- **색상 팔레트:** 활기차고 에너지 넘치는 색상 구성표를 사용합니다.
- **타이포그래피:** 표현력이 풍부한 글꼴을 사용하여 명확한 시각적 계층 구조를 만듭니다.
- **레이아웃:** 데스크톱과 모바일 장치 모두에서 사용하기 쉬운 깔끔한 중앙 집중식 레이아웃.
- **효과:**
    - 배경에 미묘한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
    - UI 요소에 깊이감을 주기 위한 다중 레이어 드롭 섀도우.
    - 버튼과 같은 상호작용 요소에 "빛나는" 효과.
- **아이콘:** 사용성과 시각적 매력을 향상시키기 위해 아이콘을 사용합니다.

### 핵심 기능
- **번호 생성:** 1과 45 사이에서 6개의 고유한 무작위 번호를 생성합니다.
- **보너스 번호:** 1개의 보너스 번호를 생성합니다.
- **표시:** 생성된 번호를 명확하고 시각적으로 매력적인 방식으로 표시합니다.
- **상호작용:** 번호 추첨을 시작하는 "생성" 버튼.
- **애니메이션:** 경험을 더 매력적으로 만들기 위해 숫자 표시를 애니메이션화합니다.

## 현재 계획

1.  **HTML (`index.html`):**
    *   애플리케이션의 기본 구조를 설정합니다.
    *   로또 기계를 위한 메인 컨테이너를 만듭니다.
    -  제목을 추가합니다.
    *   번호 표시 영역을 추가합니다.
    *   "생성" 버튼을 추가합니다.

2.  **CSS (`style.css`):**
    *   위에서 설명한 시각적 디자인을 구현하여 모든 요소를 스타일링합니다.
    *   레이아웃이 반응형이며 다양한 화면 크기에서 잘 작동하는지 확인합니다.

3.  **JavaScript (`main.js`):**
    *   고유한 무작위 번호를 생성하는 로직을 구현합니다.
    *   "생성" 버튼에 이벤트 리스너를 추가합니다.
    *   버튼을 클릭하면 새 번호로 디스플레이를 업데이트합니다.
    *   번호 공개를 위한 애니메이션을 추가합니다.
